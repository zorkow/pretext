The boundingbox and grid-axes elements then specify the basic grid layout. In our case we use a grid with labeled \(x\)-axis and \(y\)-axis which are bounded to the range from \(-4\) to \(4\) each. The following two elements are definitions for a variable and the basic function \(f(x)\text{.}\) Alone they have no graphical output, but are only referenced in later drawing commands. More precisely, graph command draws the function \(f\text{,}\) while the tangent-line element draws the tangent at the point \(f(1)\) as we have previously defined \(a=1\text{.}\) Finally we explicitly draw this tangent point with the point element, giving it a mathematical label of \((a, f(a))\text{.}\) Note, that this label is given in LaTeX code and can be of any complexity as it will later be processed and typeset by MathJax.
